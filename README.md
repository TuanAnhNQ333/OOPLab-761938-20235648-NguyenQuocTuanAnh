Java Object-Oriented Programming Lab Exercises.

This repository contains lab assignments organized by lab folders.

## Description

* **lab1 – lab5**: Lab assignments for the OOP course
* **Exercises**: Programming exercises for each lab
* **FirstProgram**: Basic Java programs and experiments
* **resources**: Project resource files

This project is managed using **Maven** and developed with **IntelliJ IDEA**.
